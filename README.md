# Gallery Mobile

A minimalist mobile companion for the NestJS Gallery API, built with Expo and React Native.

```bash
# 1. Install dependencies
npm install

# 2. Configure Environment
cp env.example .env
# Update NEXT_PUBLIC_API_URL in .env.local

# 3. Development Mode
npm run dev
```
